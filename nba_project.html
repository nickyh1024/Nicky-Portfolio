NBA Stats Prediction

Python Pandas Scikit-learn Web Deployment

This project uses NBA player season stats from 2017–2024 to train predictive models that estimate player performance in the upcoming 2024–2025 season. It incorporates contextual features like teammate influence, age effects, and league-wide trends to predict core box score statistics such as points, assists, rebounds, and more.

The final models use random forests to predict each individual stat. We then compute fantasy performance rankings based on either Points or 9-Category scoring formats.

Users can explore projected stats for top players and filter by name or team. This project is also deployed online so others can interact with it.
